# Flask File Upload App

A Flask web application for uploading files.

## Features
- Upload files
- Store uploaded files
- Display uploaded file list
- Automatic uploads folder creation

## Technologies Used
- Python
- Flask
- HTML

## Installation

pip install flask

## Run

python app.py

Open browser:
http://127.0.0.1:5000

## Purpose
focuses on file handling and upload systems in Flask.
